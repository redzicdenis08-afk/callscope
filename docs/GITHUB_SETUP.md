# GitHub setup

These settings make the repo easier to discover and trust.

Offline QA benchmark and batch analytics for AI voice-agent call transcripts.

## Topics

Set these repository topics:

```text
voice-ai, call-analytics, transcripts, vapi, retell, twilio, python, cli
```

Why: focused topics and README text help the repo land near the right builders instead of floating in generic Python soup.
