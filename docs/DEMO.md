# Demo

This is the fastest way to show why callscope is worth starring. Use fictional example data only.

## Run it

```bash
pip install -e .
python -m callscope analyze examples/*.json examples/*.txt
```

## What to screenshot

Outcome labels, customer events, conversation metrics, and score per call.

A good launch screenshot should show the command and the useful output in one image. Avoid giant terminal dumps.
